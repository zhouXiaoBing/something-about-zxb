##7.【掌握】多个对象内存管理（野指针）
* 单个对象的内存管理，看起来非常简单。
* 如果对多个对象进行内存管理，并且对象之间是有联系的，那么管理就会变得比较复杂。


* 其实，多个对象的内存管理思路，跟管理游戏房间类似：
* 如：劲舞团，LOL
```objc
    // 1、创建玩家对象
    Gamer *game = [[Gamer alloc] init];

    // 2、创建房间对象
    Room *room = [[Room alloc] init];

    // 3、让玩家进入房间
    game.room = room;

    // 4、释放房间对象
    [room release];

    // 5、输出玩家的房间
    NSLog(@"玩家的房间是：%@",game.room);
```
* 此时，会报野指针错误。
