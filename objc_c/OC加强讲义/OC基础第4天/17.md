##13、【理解】NSArray读写文件
* 1、【理解】NSArray数据写入到文件中

 ```objc
 NSArray *array = @[@"a",@"b"];
 BOOL res = [array writeToFile:@"/Users/sleepingsun/Desktop/test.txt" atomically:YES];

 NSLog(@"%@",array);
```

* 2、【理解】从文件中读取数据到NSArray中

 ```objc
 NSArray *array1 = [NSArray arrayWithContentsOfFile:@"/Users/sleepingsun/Desktop/test.txt"];

 NSLog(@"%@",array1);
```
