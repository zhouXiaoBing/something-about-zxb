##12、【理解】NSArray的遍历方法
* 遍历, 就是将NSArray里面的所有元素一个一个取出来查看

* 1、NSArray的普通遍历
```objc
for (int i = 0; i<array.count; i++) {  }
```
* 2、【理解】NSArray的快速遍历
```objc
for (id obj in array) {  }```

* 3、NSArray使用block进行遍历

 ```objc
 [array enumerateObjectsUsingBlock:^(id obj, NSUInteger  idx, BOOL *stop) {

 }];```
