7、【掌握】Protocol概念及基本使用

1、什么是Protocol?
  • 1）Protocol中文意思：协议。
  • 2）协议中只能声明一些方法。
  • 3）遵守了协议，就等于拥有了协议中的所有方法。
2、协议的基本用法：
  • 1、协议的定义格式：
    @protocol 协议名称
// 各种方法；
@end
  • 2、协议的使用格式：
    @interface 类名 : NSObject <协议名1，协议名2，...>
@end
