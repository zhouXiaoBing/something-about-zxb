5、【理解】应用：block的使用场景（二）

  • 思考：用返回值是block的函数，修改我们应用场景的代码
   // 1、重命名block
    typedef void (^workBlock)();


   // 2、block作为返回值使用
    workBlock workDay(int d){

   // 拿到循环体中的block
    workBlock work;

    switch (d) {
        case 1:
            // 真正干的实事
            work = ^{
                NSLog(@"熟悉项目代码");
            };
            break;
        case 2:
            // 真正干的实事
            work = (^{
                NSLog(@"熟悉项目代码");
            });
            break;
        case 3:
            // 真正干的实事
            work = (^{
                NSLog(@"编写代码");
            });
            break;
        case 4:
            // 真正干的实事
            work = (^{
                NSLog(@"扯淡");
            });
            break;
        case 5:
            // 真正干的实事
            work = (^{
                NSLog(@"聊天");
            });
            break;

        default:
            break;
    }
    // 返回循环体中的block内容
    return work;
}
